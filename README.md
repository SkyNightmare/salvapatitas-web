# Salva Patitas PR 🐾

**Website:** [https://salvapatitas-web.vercel.app/]  

**Description:**  
Salva Patitas PR is a website that compiles most of the animal shelters in Puerto Rico, making it easier for people to find and connect with them. The platform focuses on rescuing, rehabilitating, and rehoming dogs across the island.  

Built with **React**, **TypeScript**, and **Tailwind CSS**, this project provides a user-friendly interface for both volunteers and potential donors.  

---

## Key Features

-  **Volunteer & Donation Hub** — Browse a list of shelters and choose which ones you want to support.  
-  **Expand the Compilation** — Submit a shelter via email to have it added to the list.  

---

## Technologies Used

- **React**  
- **TypeScript (TS)**  
- **Tailwind CSS**
